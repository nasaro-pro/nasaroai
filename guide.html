← Nasaro AI로 돌아가기

1. Nasaro AI가 하는 일

비교여러 AI 답변을 한 화면에서 비교합니다.
토론주제별 AI 라운드 토론입니다.
협업작업 유형별 4단계 알고리즘 + 단계별 AI 실행입니다.
에이전트PC 확장 / Android 오버레이로 화면 위 임무 실행입니다.

2. 저장 · 프라이버시

  • 자동 기록 없음 — 비교·토론·에이전트 결과는 그 채팅 화면에만 표시됩니다.
  • 결과물 저장 — 답변 옆 「💾 결과물 저장」 버튼을 눌렀을 때만 메뉴 「결과물 파일」에 추가됩니다.
  • 비로그인 — sessionStorage에만 보관 (탭/브라우저 데이터 삭제 시 사라짐).
  • 로그인 — 서버 동기화 + Android 앱 IndexedDB 오프라인 보관.
  • 🔒 프라이버시 — 서버 동기화·공유 링크는 막지만, 수동 「결과물 저장」은 가능합니다.

3. 회원 · 관리자

  • 회원가입: /auth/signup (이메일, 비밀번호 8자+)
  • 관리자: 왼쪽 상단 「관리」→ 비밀번호 → /admin 대시보드 (회원·사용량)

4. 대중이 이 사이트를 쓰는 이유

  • ChatGPT만 쓰면 “이 답 맞나?” — 여러 AI를 한 번에 보고 고를 수 있음
  • 조사·기획·글쓰기를 토론·협업 4단계로 체계화
  • PC/폰에서 에이전트로 반복 작업 대신
  • 원하는 것만 결과물 저장 — 프라이버시 부담 적음

5. 보완 우선순위 (대중용)

  1. 베스트 답변 원탭 복사
  2. 메일·SNS·카톡 초안 템플릿
  3. 「오늘 뭐 물어볼까」 질문 추천
  4. 즐겨찾는 AI 고정
  5. 최근 질문 재실행
  6. QR로 폰↔PC 공유
  7. PDF/인쇄 한 장 내보내기
  8. 초보 모드 (기능 하나씩 크게)
  9. Android 홈 위젯
  10. 협업·에이전트 완료 알림 강화

하지 않음: REST API 키 (개발자용, 대중 불필요). 커스텀 협업 플러그인은 보류.

6. ✅ 구현됨

  • 음성·공유·다크모드·다국어·AI 프리셋·예약 에이전트
  • 결과물 저장 버튼 + 결과물 파일 메뉴
  • 회원가입/로그인 + 서버 동기화
  • 관리자 대시보드
  • Android 다단계 명령·민감 작업 확인·자동 AI
