📘 Nasaro AI 사용 가이드

처음이셔도 괜찮아요. 아래 순서대로 따라 하시면 됩니다.

시작하기

  1. 화면 아래 입력창에 하고 싶은 일을 적습니다.
  2. 비교 · 토론 · 협업 중 모드를 고릅니다.
  3. 🤖 AI 버튼을 눌러 사용할 AI를 선택합니다 (여러 개 가능).
  4. 오른쪽 ▶ 버튼으로 실행합니다.
에이전트(자동 실행)는 입력창 위 「🌐 에이전트」 버튼을 누르세요. PC는 확장 프로그램, 폰은 앱이 필요합니다.

비교 모드

같은 질문을 여러 AI에게 동시에 보냅니다. 답이 나란히 표시되어 어느 답이 나은지 바로 비교할 수 있습니다.

  • AI 칩에서 비교할 모델을 고르세요 (전체 = 6개 모두).
  • 각 답변 오른쪽 「복사」 버튼으로 텍스트를 복사할 수 있습니다.
  • 마음에 드는 결과는 💾 결과물 저장으로 보관하세요.

토론 모드

주제 하나를 놓고 AI들이 라운드별로 의견을 주고받습니다. 기획안·논점 정리에 유용합니다.

협업 모드 (4단계)

작업을 조사 → 구조 → 제작 → 검증 순으로 AI가 나눠 수행합니다.

단계별 역할

  • 1 조사 — 근거·출처·팩트 수집
  • 2 구조 — 목차·논리·작성 지시문 설계
  • 3 제작 — 실제 초안·작업물 작성
  • 4 검증 — 오류·누락 지적 (재작성 X) → 필요 시 1~3단계가 수정

사용 방법

  1. 작업 내용 입력 → AI 추천 ▶
  2. 각 단계 카드에서 AI 변경 가능
  3. ▶ 작업 시작 → 1단계부터 자동 순차 실행
  4. 완료 후 추가 수정은 입력창에 적고 추가 작업 ▶
「자기소개서」「보고서」「영상 기획」 등 요청 내용에 맞게 작업 유형이 자동 분류됩니다.

에이전트

  • PC: Chrome 확장 설치 후 「에이전트」 클릭
  • Android: APK 설치 → 오버레이·접근성 권한 허용
  • 「5분 뒤 ○○해줘」처럼 입력하면 예약 임무로 등록됩니다

저장 · 공유 · 계정

  • 자동 기록 없음 — 💾 결과물 저장을 눌러야 메뉴 「결과물 파일」에 쌓입니다.
  • 로그인하면 다른 기기에서도 결과물을 이어갈 수 있습니다.
  • 공유 링크는 로그인 후 계정으로 생성됩니다.
  • 내보내기 시 md / txt / html / pdf(인쇄) 형식을 선택할 수 있습니다.
  • 🔒 프라이버시 모드: 자동 동기화·공유 차단 (수동 저장은 가능)

자주 묻는 질문

Q. AI 답변 언어는? 질문한 언어에 맞춰 답합니다. UI 언어(설정)와 별개입니다.

Q. 하루 사용 한도? 비교 50 · 토론 10 · 협업 5 · 에이전트 20회 (KST 자정 리셋)

Q. 기록은 어디? 메뉴 「기록」= 이번 세션 요약. 영구 보관은 「결과물 저장」.

앞으로 더 좋아질 기능 · 아이디어 20

사용자 의견과 실제 사용 패턴을 반영해 순서대로 보완합니다. 완료는 이미 쓸 수 있는 기능입니다.

1
답변 복사 버튼완료
비교·토론 각 답변 옆에서 한 번에 복사
2
내보내기 형식 선택완료
md · txt · html · pdf(인쇄) 중 선택
3
협업 4단계 순차 실행완료
조사→구조→제작→검증, 단계별 실시간 확인
4
검증 AI 피드백 루프완료
검증은 지적만, 조사·구조·제작 AI가 수정
5
작업 유형 자동 분류완료
자소서·문서·영상 등 요청 내용에 맞게 협업 분류
6
에이전트 예약 임무완료
「5분 뒤 ○○해줘」→ 활동 임무에 예약 표시
7
계정 로그인·결과물 동기화완료
아이디 로그인 시 기기 간 결과물 이어하기
8
빈 화면 질문 추천완료
비교 모드에서 예시 질문 칩으로 바로 시작
9
메일·SNS·카톡 초안 변환예정
비교·협업 결과를 채널별 톤·길이에 맞게 변환
10
즐겨찾는 AI 고정예정
비교·토론·협업·에이전트마다 기본 AI 조합 저장
11
베스트 답변 하이라이트예정
비교 후 AI가 추천한 답변을 한 줄로 표시·원탭 복사
12
답변 Diff(차이만) 보기예정
여러 AI 답변에서 겹치는 부분·다른 부분만 강조
13
질문 맞춤 AI 추천예정
질문 유형(코딩·글쓰기·조사)에 맞는 AI 자동 추천
14
토론 라운드 요약 카드예정
토론 끝나면 합의·쟁점·다음 액션 한 장 요약
15
결과물 폴더·태그예정
저장한 결과물을 주제·날짜·태그로 정리·검색
16
협업·에이전트 완료 알림예정
긴 작업 끝나면 브라우저·앱 알림 (Android 우선)
17
Android 홈 위젯예정
홈 화면에서 질문 입력·에이전트 바로 실행
18
초보 모드예정
기능 하나씩 크게 보여 주는 단순 UI 전환
19
팀·가족 공유 보관함예정
로그인 계정으로 결과물을 지정 사용자와 공유
20
오늘의 질문·템플릿 갤러리예정
매일 바뀌는 추천 질문 + 자소서·기획서 등 템플릿 모음
원하는 기능 번호를 알려주시면 우선순위에 반영합니다.
