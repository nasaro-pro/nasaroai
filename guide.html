← Nasaro AI로 돌아가기

1. Nasaro AI가 하는 일

비교여러 AI 답변을 한 화면에서 비교합니다.
토론주제별 AI 라운드 토론입니다.
협업작업 유형별 4단계 알고리즘 + 단계별 AI 실행입니다.
에이전트PC 확장 / Android 오버레이로 화면 위 임무 실행입니다.

2. 계정 · 기록 저장

  • 비로그인: 비교 기록·협업 플랜·에이전트 타임라인은 브라우저/APK localStorage에만 저장됩니다. 앱/웹 데이터 삭제 시 초기화됩니다.
  • 로그인: 이메일·비밀번호 계정으로 서버(SQLite)에 동기화됩니다. 다른 기기에서도 이어갈 수 있습니다.
  • API: /auth/signup, /auth/login, /user/sync

3. 협업 처리 흐름

  1. 작업 입력 → /collab/recommend로 유형·4단계 플랜 생성
  2. 각 단계 단계 실행 → /collab/run-stage (조사=Perplexity, 구조=Anthropic, 제작=OpenAI, 검증=DeepSeek)
  3. 플랜 저장 → 로컬 + 로그인 시 서버 동기화
  4. 빠른 시작: 협업 탭 상단 템플릿 버튼 (/collab/templates)

4. 설정 · 권한 · 진단

설정 버튼을 누르면 다음을 확인·실행할 수 있습니다 (멈추지 않고 바로 설정 화면으로 연결).

  • 오버레이 권한 / 접근성 권한 (Android 앱)
  • PC 확장 설치 페이지
  • 서버 상태 · 오늘 쿼터 · 배포 SHA · 확장/APK 버전

5. PC 확장 · Android

  • 확장: 모든 탭 런처/에이전트 창, Chrome Debugger로 탭 조작
  • 새 탭: 확장 newtab.html → Google 리다이렉트 후 주입
  • Android: 오버레이 질문창 + 접근성으로 홈/앱/클릭/입력

6. 다음 보완 아이디어 (15가지 — 미구현)

  1. 음성 입력으로 에이전트 임무 말하기 (모바일·PC)
  2. 비교 결과 공유 링크 생성 (읽기 전용 URL)
  3. 팀 워크스페이스 — 협업 플랜 여러 명이 같이 편집
  4. 예약 에이전트 — 특정 시간에 임무 자동 실행
  5. 확장 설정(서버 URL·창 위치) 계정 클라우드 동기화
  6. 오프라인 캐시 — 최근 비교·협업 결과 로컬 재열람
  7. 사용자별 기본 AI 모델 프리셋 저장
  8. 에이전트 완료 시 Webhook / Telegram 알림
  9. UI 다국어 (한/영) 전환
  10. 프라이버시 모드 — 서버 로그·히스토리 최소 저장
  11. 토론·협업 결과 Markdown/PDF 내보내기
  12. 커스텀 협업 알고리즘 플러그인 (유형 추가)
  13. 비교 시작 전 429·모델 장애 사전 경고
  14. 다크 모드 테마
  15. 외부 개발자용 REST API 키 발급
