ArenaX 에이전트 설치

Chrome 확장 프로그램을 설치하면 어느 사이트에서나 AI 에이전트가 직접 웹 탐색 임무를 수행합니다.

⬇️ 확장 프로그램 다운로드 (ZIP)
Chrome 전용 · 약 20KB · 무료
⚙️ 설치 방법 (1분 소요)
1

ZIP 파일 다운로드

위 버튼을 눌러 arenax-extension.zip을 다운로드하세요. 압축을 풀지 않아도 됩니다.

📦
2

Chrome 확장 관리 페이지 열기

주소창에 아래를 입력하고 Enter를 누르세요.

chrome://extensions
🔧
3

개발자 모드 켜기

페이지 오른쪽 상단의 "개발자 모드" 토글을 켜세요.

💡 이미 켜져 있으면 다음 단계로 넘어가세요.
🛠️
4

ZIP 파일을 페이지에 드래그 앤 드롭

다운로드한 arenax-extension.zip 파일을 chrome://extensions 페이지 위로 끌어다 놓으세요. 자동으로 설치됩니다.

💡 또는 "압축 해제된 확장 프로그램 로드" 버튼 → 압축 해제한 폴더 선택도 가능합니다.
🖱️
5

권한 허용

설치 후 팝업이 뜨면 "확장 프로그램 추가"를 클릭하세요. 모든 사이트에 접근 권한이 필요합니다.

✅
6

ArenaX 사이트에서 에이전트 버튼 클릭

ArenaX로 돌아가서 🌐 에이전트 버튼을 누르면 모든 사이트에서 동작하는 런처가 나타납니다.

🚀
🎉

설치 완료! 이제 어느 사이트든 에이전트가 함께합니다.

런처(🤖)를 클릭해 임무를 지시하면 AI가 직접 웹을 탐색하며 작업합니다.
