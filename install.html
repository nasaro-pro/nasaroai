ArenaX 에이전트 설치

Chrome 확장 프로그램을 설치하면 어느 사이트에서나 AI 에이전트가 직접 웹 탐색 임무를 수행합니다.

⬇️ 확장 프로그램 다운로드 (ZIP)
Chrome 전용 · 약 20KB · 무료
⚙️ 설치 방법 (1분 소요)
1

ZIP 파일 다운로드

위 버튼을 눌러 arenax-extension.zip을 다운로드하세요. 압축을 풀지 않아도 됩니다.

📦
2

Chrome 확장 관리 페이지 열기

주소창에 아래를 입력하고 Enter를 누르세요.

chrome://extensions
🔧
3

개발자 모드 켜기

페이지 오른쪽 상단의 "개발자 모드" 토글을 켜세요.

💡 이미 켜져 있으면 다음 단계로 넘어가세요.
🛠️
4

ZIP 파일을 페이지에 드래그 앤 드롭

다운로드한 arenax-extension.zip 파일을 chrome://extensions 페이지 위로 끌어다 놓으세요. 자동으로 설치됩니다.

💡 또는 "압축 해제된 확장 프로그램 로드" 버튼 → 압축 해제한 폴더 선택도 가능합니다.
🖱️
5

권한 허용

설치 후 팝업이 뜨면 "확장 프로그램 추가"를 클릭하세요. 모든 사이트에 접근 권한이 필요합니다.

✅
6

ArenaX 사이트에서 에이전트 버튼 클릭

ArenaX로 돌아가서 🌐 에이전트 버튼을 누르면 모든 사이트에서 동작하는 런처가 나타납니다.

🚀
🎉

설치 완료! 이제 어느 사이트든 에이전트가 함께합니다.

런처(🤖)를 클릭해 임무를 지시하면 AI가 직접 웹을 탐색하며 작업합니다.

📱 모바일 안드로이드 — Kiwi Browser로 설치

iOS(아이폰)는 현재 Chrome 확장을 지원하지 않습니다. 안드로이드에서는 Kiwi Browser를 사용하면 크롬 확장을 설치할 수 있습니다.

1

Kiwi Browser 설치

Play 스토어에서 Kiwi Browser를 검색해 설치하세요. 무료입니다.

Play 스토어 바로가기 ↗
📲
2

ZIP 파일 다운로드

Kiwi Browser에서 이 페이지를 열고 위의 ⬇️ 확장 프로그램 다운로드 버튼을 탭하세요.

📦
3

확장 관리 페이지 열기

Kiwi Browser 주소창에 아래를 입력하세요.

kiwi://extensions
💡 또는 우측 상단 ⋮ 메뉴 → Extensions를 탭해도 됩니다.
🔧
4

개발자 모드 켜기

오른쪽 상단의 "Developer mode" 토글을 켜세요.
토글이 안 보이면 화면을 아래로 스크롤하거나, 페이지를 가로 방향으로 회전해보세요.

🛠️
5

ZIP으로 확장 설치

개발자 모드를 켜면 상단에 "+ (from .zip/.crx/.user.js)" 버튼이 나타납니다.
탭 → 다운로드된 arenax-extension.zip 파일 선택 → 설치 완료.

💡 파일이 안 보이면 내 파일 → 다운로드 폴더에서 찾으세요.
📂
6

ArenaX에서 에이전트 활성화

Kiwi Browser로 ArenaX를 열고 🌐 에이전트 버튼을 탭하세요.

🚀
❓ 확장 없이도 사용할 수 있나요?

확장 없이도 ArenaX의 비교 · 토론 · 협업 기능은 그대로 사용할 수 있습니다.
단, AI 에이전트(웹 자동화) 기능은 확장 프로그램이 있어야 작동합니다.
확장이 설치되지 않은 상태에서 에이전트 버튼을 누르면 이 설치 페이지로 안내됩니다.
