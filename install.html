Nasaro AI 에이전트 설치

AI 에이전트가 어느 사이트에서든 웹 탐색 임무를 직접 수행합니다.

⬇️ Android APK 다운로드
Android 8.0+ · 무료 · 자동 플로팅 버튼 앱
📲 설치 방법
1

APK 다운로드

위 버튼을 탭해 nasaroai-agent.apk를 다운로드하세요.

2

출처 알 수 없는 앱 허용

다운로드한 APK 파일을 탭하면 "출처를 알 수 없는 앱" 설치 허용 안내가 나옵니다. 허용해 주세요.

설정 → 앱 → 특별한 앱 접근 권한 → 알 수 없는 앱 설치
3

앱 설치 후 실행

Nasaro AI 에이전트 앱을 실행하고 플로팅 버튼 권한을 허용하세요.

4

에이전트 버튼 → 오버레이 질문창

앱 안의 에이전트 버튼을 누르면 화면 위에 질문창이 열립니다. 접기하면 Nasaro AI 런처가 남고, 다시 누르면 질문창이 열립니다.

오버레이 권한을 허용해야 앱 밖 화면 위에도 런처와 질문창이 표시됩니다.
5

접기/다시 열기

질문창에서 접기를 누르면 Nasaro AI 런처만 남습니다. 런처를 누르면 질문창이 다시 열리고, 종료를 누르면 둘 다 사라집니다.

⬇️ 확장 프로그램 다운로드 (ZIP)
Chrome · Edge · Brave 지원 · 무료
⚙️ 설치 방법
1

ZIP 다운로드

위 버튼으로 nasaroai-extension.zip을 받으세요. 압축을 풀지 않아도 됩니다.

2

Chrome 확장 관리 페이지 열기

주소창에 입력 후 Enter

chrome://extensions
3

개발자 모드 켜기

오른쪽 상단 "개발자 모드" 토글을 켜세요.

4

ZIP 파일 드래그 앤 드롭

nasaroai-extension.zip을 확장 관리 페이지 위로 끌어다 놓으세요.

또는 "압축 해제된 확장 프로그램 로드" → 압축 해제 폴더 선택
5

Nasaro AI에서 에이전트 버튼 클릭

Nasaro AI로 돌아가 🌐 에이전트 버튼을 누르세요.

🏪 Chrome 웹 스토어 출시 예정

등록 후에는 개발자 모드 없이 한 클릭으로 설치됩니다.

⚠️ iPhone/iPad에서 APK는 설치 불가합니다
APK는 Android 전용 설치 파일입니다. iOS(iPhone·iPad)는 .ipa 형식만 지원하며, App Store 외 경로 설치는 Apple이 제한합니다.

또한 iOS의 모든 브라우저는 Apple의 정책으로 인해 Chrome 확장 프로그램을 지원하지 않습니다.
(Safari, Chrome iOS, Firefox iOS 모두 동일)
📱 iOS에서 현재 가능한 것
✓

Nasaro AI 웹 앱 (PWA) 지금 가능

Safari에서 Nasaro AI를 열고 하단 공유 버튼 → "홈 화면에 추가"를 탭하세요.
홈 화면에 아이콘이 생기고, 앱처럼 전체 화면으로 사용 가능합니다.

비교 · 토론 · 협업 기능 모두 사용 가능.
단, 에이전트(웹 자동화) 기능은 사용 불가.
