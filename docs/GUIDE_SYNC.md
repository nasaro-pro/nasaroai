# 가이드 동기화 규칙

기능·구조·배포를 수정할 때 **`guide.html`도 같은 PR/커밋에서 함께** 업데이트합니다.

## 반드시 갱신하는 섹션

1. **기능 설명** — 비교/토론/협업/에이전트/계정 동작이 바뀌면 해당 문단 수정
2. **처리 흐름** — API·클라이언트·Android·확장 흐름 변경 시 반영
3. **보완 아이디어 15개** — 구현한 항목은 목록에서 빼고, **새로운 미구현 아이디어**로 교체

## 체크리스트

- [ ] `guide.html` `/guide` 경로에서 열리는지
- [ ] 사이드메뉴 **구조·아이디어 가이드** 링크 유효한지
- [ ] 설정 → **배포·버전**에 표시되는 확장/APK 버전과 `manifest.json` / `build.gradle` 일치하는지
- [ ] 계정·동기화·쿼터·권한 바로가기 설명이 실제 UI와 맞는지

## 파일 위치

- 사용자용: `guide.html` (웹 `/guide`)
- 개발용 메모: `docs/nasaroai-overview.md` (상세 아키텍처, 선택 갱신)
